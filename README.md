## Quick Start

**1. Install the plugin**
```
/plugin marketplace add MasslessAI/push-claude-plugin
/plugin install push-todo@push-claude-plugin
```

**2. Connect your account**
```
/push-todo setup
```

That's it! You'll see your tasks when you start Claude Code.

## Commands

| Command | What it does |
|---------|--------------|
| `/push-todo` | See your tasks |
| `/push-todo setup` | Connect account or fix issues |
| `/push-todo #427` | Jump to a specific task |

## Requirements

- [Push iOS app](https://pushto.do) (free)
- Claude Code
